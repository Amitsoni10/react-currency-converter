# 💱 React Currency Converter

A modern and responsive Currency Converter application built with **React**, **Vite**, and **Tailwind CSS**. The application allows users to convert currencies using real-time exchange rates through API integration.

---

## ✨ Features

- 🌍 Real-time currency conversion
- 💱 Convert between multiple currencies
- 🔄 Swap currencies with a single click
- 🔍 Easy currency selection
- ⚡ Fast and optimized performance with Vite
- 📱 Responsive design for desktop and mobile devices
- 🎨 Clean and modern UI built with Tailwind CSS
- 🔗 Live exchange rate API integration

---

## 🛠️ Tech Stack

- **React.js**
- **Vite**
- **Tailwind CSS**
- **JavaScript (ES6+)**
- **REST API**

## 🚀 Getting Started

Follow the steps below to run the project locally.

### 1. Clone the Repository

```bash
git clone https://github.com/Amitsoni10/react-currency-converter.git
```

### 2. Navigate to the Project Directory

```bash
cd react-currency-converter
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Development Server

```bash
npm run dev
```

The application will start running at:

```text
http://localhost:5173
```

Open the URL in your browser to view the application.
